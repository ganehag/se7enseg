se7enseg.js

Simple library to handle 7-segment displays on a web page

Example #1

Features

  • No images.
  • Only CSS (JS only needed to make it work).
  • Themeable
  • Resizable*
  • Alpha Numerical. 0-9, a-z, A-Z
  • BSDv3 License.
*but probably not in the way you want it to be

Gotchas

  • Requires jQuery
  • Though not confirmed. Using this library, will make a lot women fell like they want to sleep with you. So if you are not into that sort of thing. Be careful!

Usage

Include jQuery and this library into your project. Having a taco is also a good idea. But completely optional. 

<script src="//ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
<script src="se7enseg.min.js"></script>

Example #2

Se7enSeg can be loaded into any block element. The most common use-case is to load it with the option blocks. 

// HTML part
<div id="foo" data-text="Hello, world"></div>

// JS part
$('#foo').se7enseg({blocks: 11});

Example #3

Or if you want to use a smaller size. Keep in mind that since size is based on em. It will be hard to get an exact match in a px sized container. 

// HTML part
<div id="bar" data-text="abcdefghijklmnopqrstuvwxyz"></div>

// JS part
$('#bar').se7enseg({blocks: 26, size: '3pt'});

Example #4

Or if you wish to change the default colors to something cooler. 

// HTML part
<div id="bar" data-text="ABCDEFGHIJKLMNOPQRSTUVWXYZ"></div>

// JS part
$('#bar').se7enseg({blocks: 26, size: '5pt', off: '#d1eefb', on: '#4ca1c9'})

Options

options: {
    blocks: 4,              // Number of 7-segment blocks
    cssClass: 'se7enseg',   // If you want the default class to be something else
    soft: false,            // Enable slightly rounded corners on all the segments
    slant: false,           // Enable slanted blocks
    size: '9pt',            // Base size for the block (not the same as actual size)
    off: undefined,         // Color in CSS format for the 'off' state
    on: undefined           // Color in CSS format for the 'on' state
}

API

// Load into target
$(/* target */).se7enseg({/* options */});

// Send a new string to the display
$(/* target */).se7enseg('write', 'Message');

The library uses the data attribute when loading a new display. If the element contains a data-text attribute. The content of this attribute will be display at load time. As can be seen in example #2, #3 and #4.

Wrapping display

The content will wrap if the width is not large enough.
